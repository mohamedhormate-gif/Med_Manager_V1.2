# Med_Manager_V1.2
Cette fonctionnalité permet de rechercher dans une liste de `Personne` via un terme appliqué au nom, prénom ou ID. La recherche est insensible à la casse, supporte les correspondances partielles et gère les valeurs nulles. Elle retourne une liste de résultats, assurant modularité, réutilisation et séparation entre logique et affichage.
